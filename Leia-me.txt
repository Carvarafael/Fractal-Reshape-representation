----------------------PARTE 1 - SCRIPTS------------
Nesta pasta, estão os scripts usados para extração de atributos, classificação e outras pequenas funções. A maioria é pra Matlab.

-Atributos fractais
	*ScriptPercCBM: usado para obter atributos de percolação usando a distância de Minkowski (chessboard), como publicado no artigo da CBM de 2017
	*ScriptPercLACDF3Distances: usado para obter as três métricas fractais usando os três tipos de distâncias
	*SaveCSVPercLACDF3Distances: usado para salvar em formato CSV (com cabeçalho .arff) os atributos obtidos com ScriptPercLACDF3Distances
	*Outros scripts do diretórios: são sub-funções dos scripts anteriores

-Outros atributos handcrafted
	*ScriptLBPFeatures: obtém atributos LBP de imagens de um dataset
	*SaveLBPFeaturesCSV: salva os atributos obtidos com ScriptLBPFeatures em formato CSV (com cabeçalho .arff)
	*ScriptHaralick: obtém descritores Haralick de imagens de um dataset
	*SaveHaralickFeaturesCSV: salva os atributos obtidos com ScriptHaralick em formato CSV (com cabeçalho .arff)
	*haralickTextureFeatures: sub-função de ScriptHaralick

-Outros scripts
	*ADASYNScriptForRGB: gera imagens sintéticas usando a técnica ADASYN
	*Benign (1).png: imagem meramente ilustrativa
	*cerecurr_y: sub-função de reshapeRecPlot
	*confusionmatStats: script para obter métricas a partir de uma matriz de confusão
	*Create10Folds: gerar 10 folds treino/teste para classificar imagens em uma CNN
	*Create10x10FoldsAllTypesImages: gera 10 combinações diferentes de 10 folds treino/teste para os 4 diferentes modelos de representação
	*NewensembleFNN: script para criar um ensemble
	*plot_gaf: script para gerar imagens GASF e GADF. Trata se de um notebook python. Sugiro rodar no Google Colab
	*reshapeRecPlot: script para gerar imagens usando plot de recorrência
	*SingleCNN10folds: classificação via CNN usando 10-folds cross validation
	*SingleCNN10foldsGradCAM: mesmo que o acima, porém também gera mapas de calor pra cada imagem teste usando Grad-CAM

----------------------PARTE 2 - matlab2weka------------
Nesta pasta estão todas as funções necessárias para usar o Weka via Matlab. Porém, caso queira apenas fazer classificações simples de atributos handcrafted, basta usar os dois arquivos no diretório principal

	*wekaClassification: realiza classificações de atributos handcrafted no Weka usando a interface o Matlab
	*weka.jar: arquivo necessário para a execução do script acima. Também é necessário ter o Weka instalado na máquina

----------------------PARTE 3 - Imagens Reshape------------
Contém as imagens gerada a partir das técnicas de reshape sequencial (reshape comum), plot de recorrência, GASF e GADF para os datasets CR, LA, LG, NHL e UCSB. Também contém as imagens originais e as imagens em escala de cinza usadas para geras as GASF e GADF. Recomendo fortemente que NENHUM arquivo ou diretório desta parte sejam renomeados.

----------------------PARTE 4 - FoldsSetAll------------
Esse diretório contém os imgDataStore separados em 10 grupos de 10 folds que apontam para o local das imagens dos datasets. Também contém outros arquivos necessários para a construção dos ensembles como os TrueFoldsSetX.mat. ATENÇÃO, renomear os arquivos do diretório Imagens Reshape pode acarretar em erros nesta parte.
